What you need: A free Google account (for Firebase) and a free GitHub account. That's it — everything else is free and takes about 15 minutes total.
1
Set up Firebase (your free database + login)
Takes ~5 minutes • Free forever on Spark plan
1
Go to Firebase Console
Open console.firebase.google.com and sign in with your Google account.
2
Create a new project
Click "Add project" → Name it eid-gift-box → Disable Google Analytics (not needed) → Click "Create project"
3
Add a Web App & get your config keys
In your project dashboard, click the </> Web icon → Give it a nickname like eid-gift-box → Click "Register app".

You'll see a config object like this — copy all the values:
javascript — copy these values 
// Firebase gives you something like this:
const firebaseConfig = {
  apiKey:            "AIzaSyB...",        ← copy this
  authDomain:        "eid-gift-box.firebaseapp.com",
  projectId:         "eid-gift-box",
  storageBucket:     "eid-gift-box.appspot.com",
  messagingSenderId: "123456789",
  appId:             "1:123456789:web:abc123",
};
4
Paste values into src/firebase.js
Open the src/firebase.js file and replace the placeholder text with your real values:
src/firebase.js
const firebaseConfig = {
  apiKey:            "YOUR_REAL_API_KEY",     ← paste here
  authDomain:        "YOUR_REAL_AUTH_DOMAIN",
  projectId:         "YOUR_REAL_PROJECT_ID",
  storageBucket:     "YOUR_REAL_BUCKET",
  messagingSenderId: "YOUR_REAL_SENDER_ID",
  appId:             "YOUR_REAL_APP_ID",
};
5
Enable Firestore Database
In the left sidebar: Build → Firestore Database → "Create database" → Choose "Start in test mode" → Pick any location → Done
⚠️
Test mode allows all reads/writes for 30 days. After that, paste the contents of firestore.rules in the Rules tab to secure your database.
6
Enable Authentication
In the left sidebar: Build → Authentication → "Get started" → Click Email/Password → Toggle it Enable → Save
✅
Firebase is ready! Your database and login system are set up. On to GitHub.
2
Put your code on GitHub
Vercel deploys directly from GitHub — free forever
1
Create a GitHub account (if you don't have one)
Go to github.com → Sign up free. Takes 1 minute.
2
Create a new repository
Click the "+" button top-right → "New repository" → Name it eid-gift-box → Keep it Public → Click "Create repository"
3
Upload your project files
On your new repository page, click "uploading an existing file" link, then drag and drop the entire eid-gift-box folder contents. Make sure your file structure looks like this:
eid-gift-box/ ├── package.json ├── firestore.rules ├── public/ │ └── index.html └── src/ ├── index.js ├── App.js ← with your Firebase values filled in └── firebase.js ← with your config values
Scroll down → Click "Commit changes"
✅
Your code is on GitHub! Now Vercel will host it for free.
3
Deploy to Vercel (get your shareable link)
Takes ~3 minutes • Free forever • Auto-updates when you change code
1
Sign up for Vercel
Go to vercel.com → Click "Sign up" → Choose "Continue with GitHub" → Authorize Vercel
2
Import your repository
On your Vercel dashboard, click "Add New Project" → Find eid-gift-box in the list → Click "Import"
3
Configure the project — no changes needed!
Vercel automatically detects it's a React app. The settings should be:
Vercel settings (auto-detected)
Framework Preset:  Create React App  ← auto-detected ✓
Build Command:     npm run build       ← auto-filled ✓
Output Directory:  build               ← auto-filled ✓
Install Command:   npm install         ← auto-filled ✓
Just click "Deploy" — no other settings needed!
4
Wait ~2 minutes for deployment
Vercel builds and deploys your app. You'll see a progress screen. When it finishes you get a screen like this:
VERCEL DEPLOYMENT SUCCESS
🎉
Deployment successful!
Your site is live at: eid-gift-box-yourname.vercel.app
🎉
You're live! Vercel gives you a URL like eid-gift-box-xyz.vercel.app — copy it and share it with your friends right now!
4
Optional: Custom domain name
e.g. eid-gift.com — costs ~$10/year from Namecheap
If you want a cleaner URL like mygiftbox.com instead of eid-gift-box-xyz.vercel.app:
1
Buy a domain (optional)
Go to namecheap.com → Search for a domain → Buy it (~$10/year for .com)
2
Add it to Vercel
Vercel Dashboard → Your project → Settings → Domains → Add your domain → Follow the DNS instructions → Done in ~5 minutes!
💡
The Vercel URL (eid-gift-box-xyz.vercel.app) works perfectly fine and is completely free! A custom domain is just for style.

Troubleshooting
Common issues and fixes
!
Build fails with "firebase not found"
Make sure package.json has "firebase": "^10.7.1" in dependencies. Vercel runs npm install automatically so this should install it.
!
Login doesn't work / auth error
Make sure you enabled Email/Password in Firebase Authentication. Also check that your firebase.js config values are correct (no extra spaces).
!
Firestore permission denied
Go to Firebase Console → Firestore → Rules → Make sure it's in test mode (allow read, write: if true) or paste the rules from firestore.rules.
!
AI message generator doesn't work on deployed version
The AI feature uses the Anthropic API which requires a key. In the deployed version it will fall back to built-in preset messages automatically — everything else works perfectly!
